# Inlove.postil — Landing

Bed linen website on React + Vite.

## Quick Start

```bash
# 1. Go to the project folder
cd inlove-postil

# 2. Install dependencies
npm install

# 3. Run in development mode
npm run dev
# → open http://localhost:5173

# 4. Build for production
npm run build
```
